# BREAST-CANCER-PREDICTOR
The goal of this project is to build a model that can accurately classify breast cancer cases as Malignant or Benign based on medical features. It uses Data preprocessing and cleaning using Pandas and Numpy. Model building using Logistic Regression and K-Nearest Neighbors(KNN). A simple and interactive web-based application.
